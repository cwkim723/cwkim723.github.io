Decakill information

  • 개발일정:
    2022.04 ~ 2022.10
  • 한줄소개: 스트리머 플레이 예측을 통해 사람들과 경쟁하고 보상을 받아갈 수 있는 트위치 익스텐션(데카킬)의 웹페이지
  • URL: https://decakill.com
  • 특이사항: 서버 종료
  • 역할 및 기능개발:
    • 데카킬 웹 페이지 백엔드 담당
    • Kotlin, Spring Boot(Gradle), JPA, QueryDSL, MongoDB, Redis, MySQL
    • 트위치 로그인, gmail(smtp)이용 메일 발송(+이미지 발송 가능), 스토어(응모, 구매), 랭킹(스트리머, 이스포츠(현재 hide)랭킹, 데일리랭킹) 등 구현

메인 페이지

메인

  • 데카킬과 파트너쉽을 맺은 스트리머가 생방송 중일 경우 On Air 표시
    •

    랭킹

    전체 랭킹

  • 전체 랭킹은 여태까지 데카킬 캠페인 데이터를 총 합산한 결과를 나타냄
  • 100위까지 나타냄
    •
    랭킹 검색

  • 닉네임을 입력하면 데이터가 존재할 경우 검색된 데이터가 나타난다.
    •
    데일리 랭킹

  • 데일리 랭킹은 당일 게임의 랭킹이 제공됨(스트리머 별)
    •

    로그인

    트위치 로그인

  • 로그인은 트위치 로그인만 제공
  • 계정을 입력하고 권한 동의를 하면 로그인이 완료된다.
  • 회원이 아닌 경우에는 회원가입 권한동의 창으로 이동
    •

    스토어

    스토어 목록 - 비로그인 시

  • 데카킬 예측에 참여한 유저들은 젬이라는 데카킬 포인트를 획득하고 젬을 모아 물품을 구매할 수 있음
  • 구매 혹은 응모를 통해 상품을 구매할 수 있고, 구매가 가능한 경우 구매가능 이라는 표시가 뜸
    •
    스토어 목록 - 로그인 시

  • 로그인 시 본인의 젬 정보가 나타남
    •

    마이페이지

    나의 데이터

  • 젬(포인트) 정보와 데카킬에 참여한 기록을 보여 줌
    •
    획득 내역

  • 젬(포인트)을 어떤 스트리머의 방송에서 획득했는지를 나타냄
  • 스토어에서 상품 환불 시의 내역도 나타냄
    •
    사용 내역

  • 젬(포인트)을 사용했을 경우 (스토어에서 상품 구매 / 응모한 경우) 기록을 보여 줌
    •
    나의 정보

  • 이메일 인증 및 개인 정보 입력 페이지
    •
    1:1 문의
