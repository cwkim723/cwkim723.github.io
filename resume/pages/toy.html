자바스크립트 챌린지 information

  • 개발일정:
    2022.02
  • 한줄소개: 노마드 코더 사이트 내 자바스크립트 챌린지 참여
  • 역할 및 기능개발:
    • 로컬 스토리지를 사용하여 닉네임 입력을 통한 로그인 구현
    • 로컬 스토리지를 이용한 Todo List 기능 구현, 해야할 목록과 완료한 목록을 구분함으로써 사용에 편의를 줌
    • 현재 시각을 나타내는 시계 구현
    • 여러 SF 영화 추천이 랜덤으로 나타나도록 구현
    • 날씨 API를 통해 현재 위치를 받아 해당 장소의 날씨를 알려주는 기능 구현
    • 새로고침을 하면 배경이 랜덤으로 나타나도록 구현

자바스크립트 챌린지 플로우

기능 목록
  • 로그인
  • 배경화면
  • 영화
  • 투두리스트
  • 날씨

로그인

로그인 페이지

  • 닉네임을 입력하고 로그인 버튼을 누르면 메인화면으로 진입할 수 있다.
  • 입력하지 않은 경우 입력하라는 경고 메시지가 뜬다.
    •
    메인 페이지 접근

  • 로그인을 하여 메인 페이지에 접근하였다. 로그인에 사용했던 닉네임이 화면에 나타난다.
    •

    배경화면

    배경화면 랜덤 변경

  • 우주를 배경으로 한 3개의 배경화면이 랜덤으로 나타난다.
    •

    영화추천 랜덤 변경

  • 새로고침을 할 때마다 SF 영화가 랜덤으로 나타난다.
  • 영화제목, 감독, 출연진, 줄거리, 포스터 정보가 보여진다.
    •

    투두리스트

    투두리스트

  • 오늘 할 일을 적고 엔터를 누르면 해야할 목록에 등록이 된다.
  • 체크 표시(✔️)를 누르면 완료한 목록으로 이동한다.
  • 엑스 표시(❌)를 누르면 항목이 삭제된다.
    •

    날씨

    날씨 API

  • https://api.openweathermap.org의 API를 사용하여 위치 허용 시 현재 날씨를 보여준다.
